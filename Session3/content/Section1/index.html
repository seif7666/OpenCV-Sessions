Line Detection techniques

As we said, don't apply the following algorithms before you manage to successfully process the data.

Method 1 : Hough Lines

This method operates using the polar coordinates (using rho and theta). There are multiple parameters.

The following parameters must be tuned to obtain required performance:

  • Set resolution for rho and theta.
  • Set a threshold for which each line must be detected more than it to be detected as line.
  • Optional: Set angle intervals

The return of the function is a list of points in polar coordinates.

Method 2 : Hough Lines Transform

Relative to first method, we can see it is slower. But it can produce more optimistic results than the first method since it :

  • Cancels noise better.
  • Enables you to specify the length of line and distance between each point.
