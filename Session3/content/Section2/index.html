Circle Detection

Circle detection might be a hard step, since it needs delicate processing.

Hough Circles method is used for detecting circles but few parameters should be adjusted:

  • Method

    You can choose between several methods for circle detection which are :

    1. cv2.HOUGH_GRADIENT
    2. cv2.HOUGH_GRADIENT_ALT : This is seen to get better results.
  • dp: (Inverse Ratio Resolution), reccomended:1.5
  • param1: The value of highest threshold for Canny edge detection, the lower is calculated as half this value. Choose high values that may reach 300 or more.
  • param2: The Accumulator threshold. Same as in hough lines
  • Min Radius
  • Max radius

    Note: putting values <=0 means setting larger radius to be size of image, but if method is HOUGH_GRADIENT, then no radii are detected, only centers
