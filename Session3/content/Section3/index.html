Image Contours

Contours are simply represented as the boundaries of an object in an image. To find these contours, the image should be processed and canny edge detector is applied on it.

Finding Contours

cv2.findContours is a very important function. By the name it is used to detect all contours in an image. The function takes the following parameters:

  1. Image
  2. Contour Retrieval Mode and they are:
    • RETR_EXTERNAL: Retrieves Extreme Outer contours
    • RETR_LIST: Retrieves all contours without specifying any relationship
    • RETR_CCOMP:Split the contour to outer contours and inner.
    • RETR_TREE:Put the contour as a tree hierarchy.
