Task

You will be handed several videos for a transect line. The transect line image consists of the following:

  • 2 Blue lines
  • 9x3 squares. Each square may contain a
    • Red Star
    • Yellow Square
    • Sponge
    • Coral Reef
    • None

It is required to analyze a row of squares (3 adjacent squares), and state the object in each one whether it is

  • Red start
  • Yellow Square
  • Otherwise (which might be sponge, coral reeg or just nothing.

To get the image of row of squares use mouse callback to crop the image. Just specify the height but take the whole image width

You will be handed some videos to view the shapes and some images to test your output on them.
