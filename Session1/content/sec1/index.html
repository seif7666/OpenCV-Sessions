How image is represented

Loading an image

To load an image, the function cv2.load is used. The second argument has the value zero which indicates that we want to read the image in gray scale format.

Recognizing the image features

From running the following cells we can see that:
  1. Image is represented as a numpy array.
  2. Image is represented as 2D matrix
  3. The datatype of the matrix is uint8 which represents integer values from 0 to 255

But what does the numbers in the matrix mean?

  1. The picture is represented as X*Y pixels.
  2. Each pixel is represented by a number.
  3. This number represents the color of the pixel.

Binary Image

A binary image is an image that only carries 2 values, either 255 for white or 0 for black, it can be represented as a boolean matrix
