Open CV is an open source library that has Python and C++ distributions and it is used in image processing and computer vision tasks.

Session Contents

In this session we will discuss the following topics
  1. How an image is represented
  2. Doing operations on the image
    • Doing some mathematical operations
    • Image partitioning
  3. Applying operations on multiple images
    • Getting difference
    • Image concatenations
    • Feature Mapping
  4. Image thresholding
  5. Different filters
  6. Edge detection
