Color Extraction

Now, if we want to show only certain colors in an image. Then we need to clearly specify the range of the color. Identifying this is much easier in HSV than RGB.

This process is simple in OpenCV, where you perform the following steps:

  1. State the minimum color value and max color value
  2. Read image
  3. Change color space to HSV using cv2.cvtColor
  4. Apply minimum and maximum range on image using cv2.inRange. Make sure to visualize the output mask.
  5. Perform bitwise operation on image using OpenCV bitwise operations.

Task

Create minimum and maximum bounds trackbars that control which color to be extracted on a running video capture.
