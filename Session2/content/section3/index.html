Color Segmentation

Color Segmentation means to partition our image into multiple segments where each segment represents a certain color.

Unsupervised Learning

In Machine Learning, we have 3 major fields and they are :

  • Supervised Learning

    Where training data has an output cluster, target or label. The required task here is to classify these data correctly.

  • Unsupervised Learning

    Training data don't have labels, the required task is to cluster the data.

  • Reinforcement Learning

In Unsupervised Learning, we need to cluster our image to different culsters each with certain dominating color. This could be accomplished using a famous algorithm called Kmeans.
