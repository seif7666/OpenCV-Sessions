Colors Representation

Color Representation types

Before we start talking about how an image is represented, we should know how a specific color is represented. We have 3 ways to represent colors, which are :

  • RGB representation

    (RGB) refers to (Red, Green, Blue).Color space consists of 3 numbers ranging from 0 to 255 determining the intensity of each color in the pixel.

    • Color with largest intensity is white with intensity (255,255,255)
    • Color with lowest intensity is black with intensity (0,0,0)
    • The largest intensity of each color is (255,0,0), (0,255,0), (0,0,255)

  • HSV representation

    (Hue Saturation Value color space. The values ranges from (0:360,0:100,0:100)

  • HSL representation
